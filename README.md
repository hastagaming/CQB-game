### ✨ Fitur Terbaru:
* **Minecraft Style Controls**: D-Pad khusus untuk pengguna HP agar pergerakan lebih presisi.
* **Tactical Interior**: Jelajahi ruangan rumah dengan sistem tabrakan (collision) yang akurat.
* **Multiplayer Ready**: Main bersama teman menggunakan sistem PeerJS (P2P).
* **Hybrid Input**: Otomatis mendeteksi penggunaan Keyboard (PC) atau Touchscreen (HP).

### 🛠️ Cara Bermain:
1. Jika di **PC**: Gunakan **WASD** untuk bergerak dan **Mouse** untuk menembak.
2. Jika di **HP**: Gunakan **D-Pad** di kiri dan tombol **FIRE** di kanan.
3. Untuk Multiplayer: Bagikan ID kamu yang muncul di pojok kiri atas ke temanmu.
